Midjourney V5 : How to use it ? The guide

MidJourney


Midjourney is an research lab that produces an artificial intelligence program that creates images made from textual descriptions,

like OpenAI's DALL-E.

Website

Pricing : Free for 25 images / month

Midjourney is an AI-powered image generator that creates artistic visuals based on concise textual descriptions. Its V5 version brings forth enhanced realism and intricate details to the generated images.

Similar to Dall-E, developed by OpenAI, the same organization behind ChatGPT, Midjourney utilizes artificial intelligence and machine learning to generate images. However, unlike Dall-E, Midjourney focuses more on the artistic aspect of creation. The recently announced V5 version further improves upon the already astounding capabilities of this AI image generator.

Midjourney V5 enhances the recipe through incremental optimizations, rather than revolutionary changes, resulting in an overall improved user experience.

  • It offers a significantly broader range of stylistic options, more attuned to various artistic influences.
  • The image quality is substantially higher, with a doubled resolution and improved dynamic range. The level of detail in the images is enhanced, making them more likely to be accurate.
  • The responsiveness to prompts is improved, ensuring a smoother interaction.
  • Additionally, users can now request refined textures without visible seams.

Midjourney V5 also supports image ratios exceeding 2:1, although this feature is still experimental.

Wondering how to use Midjourney with Discord?


The image generator is accessible to the public through a Discord server and various channels.

To get started, you'll need a Discord account and join the Midjourney server.


To access the settings, use the "/settings" command in any channel.

  • Join the Midjourney Discord server.
  • Join any channel with "newbies" in its name, indicating it's for beginners.
  • Type the command "/settings" and hit enter.
  • Select "MJ version 5" from the options.
  • Type the command "/imagine" followed by a description of the desired image.

Alternatively, you can use the command "/imagine" followed by the image description and end it with "--v 5" (optionnal if you already toggle it on in the settings).

Midjourney V5 is now available.

So, how can you make the most of it?

The free trial period is coming to an end, and Midjourney encourages users to subscribe to a paid plan starting at $10, $30, or $60 per month.

Subscribers gain access to increased GPU time for generating images and receive various escalating benefits based on their chosen plan.

If you're curious to see examples of images generated by Midjourney V5, you can find them on Twitter using the hashtag #midjourneyv5.
